# Estrutura try-catch

- **Bloco try**
  - Contém o código que representa a execução normal do trecho de código que pode acarretar uma exceção
- **Bloco catch**
  - Contém o código a ser executado caso uma exceção ocorra
  - Deve ser especificado o tipo de exceção a ser tratada (upcasting é permitido)

## Sintaxe

```java
try {

}
catch (ExceptionType e) {

}
catch (ExceptionType e) {

}
catch (ExceptionType e) {

}
// Pode ter um ou mais blocos catch
```
