# Lendo arquivo texto com classes File e Scanner

## Classes

- **`File`** - Representação abstrata de um arquivo e seu caminho
- **`Scanner`** - Leitor de texto
  - O mesmo scanner para ler dados do terminal/console.
- **`IOException`** (`Exception`) - o compilador obriga para que a exceção seja tratada
  - Exceção para erro de entrada/saída
